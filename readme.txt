To-Do List
This repository contains a simple To-Do List application built using HTML, CSS, and JavaScript. It allows you to add, remove, and mark tasks as completed.

Installation
To use this application, you can either download or clone the repository to your local machine. To clone the repository, use the following command:

bash
Copy code
git clone https://github.com/velaspis/to-do-list.git
Once you have the code on your machine, you can open the index.html file in your web browser to use the application.

Usage
To add a new task to the list, simply type it into the input field at the top of the page and press the "Add" button. The new task will appear at the bottom of the list.

To mark a task as completed, click the checkbox next to it. The completed task will be visually distinguished from the other tasks.

To remove a task from the list, click the "X" button next to it.
